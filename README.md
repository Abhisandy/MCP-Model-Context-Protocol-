We can Run this in 3 Different Ways.

1. Terminal: uv run server/client.py
2. MCP Inspector: uv run mcp dev server/weather.py
3. Using Claude App: uv run mcp install server/weather.py  (and then using your Tool in Cluade App after adding config)
